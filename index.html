My Projects

GitNote

A notes app for Android backed by your own Git repo

Design inspried by evernote, though currently only basic functionality is provided. Provides automatic sync across multiple devices. Rather than some 3rd party having your notes, they are kept in your personal git repo, wherever that may be.

This was quickly hacked together in June 2013 and hasn't really been touched since, its on github if anyone wants to continue the project.

RaceTrace

An Android app for tracking race participants

Mount your phone to your bike and keep up to date with the positions of the other riders.

Provides real-time location updates of all race participants. Set up sessions using either bluetooth or the internet. Uses either peer to peer communication or a central server.

This was my final year project in year 3 of the Cambridge Computer Science Tripos. It is functionally complete but not in any way polished. The associated dissertation is available here, and the source is on github.
